## Fix vidéo TikTok corrompue
- MoneyPrinterTurbo génère parfois des MP4 avec "moov atom not found" (fichier incomplet)
- Solution : re-encoder manuellement avec FFmpeg :
  ```
  ffmpeg -y -i combined-1.mp4 -i audio.mp3 -c:v libx264 -preset fast -c:a aac -b:a 192k -shortest -movflags +faststart output.mp4
  ```
- `-movflags +faststart` place le moov atom au début pour lecture streaming
