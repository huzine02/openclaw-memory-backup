# Analyse TikTok Viral — Ce qui marche VRAIMENT (février 2026)

## 🔴 PROBLÈME MAJEUR AVEC NOS VIDÉOS ACTUELLES

Nos vidéos sont du **faceless stock footage** (Pexels) avec voix off TTS. C'est le format le MOINS performant sur TikTok en 2026.

### Pourquoi ça ne marchera PAS :
1. **TikTok favorise l'authenticité** — les vidéos "raw" surperforment les vidéos éditées de 3-5x
2. **Faceless = 0 connexion** — l'algo pousse les vidéos avec un VISAGE (temps de rétention ×2)
3. **Voix TTS = signal de spam** — TikTok détecte et pénalise les voix synthétiques
4. **Stock footage = déjà vu** — Pexels a été scrappé par 1000 comptes similaires
5. **55% des users sont des hommes, 76% ont 18-24 ans** — notre cible (40+) est MINORITAIRE sur TikTok

## ✅ CE QUI CARTONNE VRAIMENT EN 2026

### Format #1 : Talking Head (90% des virals santé/nutrition)
- **Quelqu'un parle à la caméra, point**
- Exemples FR : Dr Jimmy Mohamed (3.7M), Alex Levand (2.4M), Tibo InShape (17M)
- Pourquoi : confiance, authenticité, rétention
- **Stats** : 2-5x plus de vues qu'un montage stock

### Format #2 : Green Screen + Screenshots
- Fond vert avec screenshot de Google/étude/article
- Le créateur commente par-dessus
- Très populaire dans la niche santé/médical

### Format #3 : Before/After avec texte overlay
- Transformation réelle, pas stock
- Texte gros au centre
- Musique tendance (pas BGM random)

### Format #4 : "POV" / Storytime
- Caméra selfie, storytelling personnel
- "POV: ton médecin te dit que t'es prédiabétique"
- Très émotionnel = très partagé

## 🎯 CE QU'ON DOIT FAIRE (recommandation)

### Option A : Hocine se filme (BEST ROI)
- iPhone, éclairage naturel, 15-30 secondes
- Script mémorisé (pas lu)
- Hook dans les 2 premières secondes
- Montage rapide (cuts tous les 3-5s pour rythme)
- **C'est ce que font TOUS les créateurs qui cartonnent**

### Option B : UGC Creator payé (moyen terme)
- Trouver un créateur FR fitness/santé sur Fiverr/Malt
- 20-50€ par vidéo
- Il/elle filme le script qu'on écrit

### Option C : Améliorer le faceless (dernier recours)
- Remplacer Pexels par des clips plus "cinématiques"
- Utiliser une VRAIE voix humaine (ElevenLabs clone, pas Edge TTS)
- Ajouter des effets "natifs TikTok" (zoom in/out, transition son)
- Format "reddit story" ou "life hack" plutôt que narration

## 📊 HOOKS QUI MARCHENT (analysés sur top performers)

### Pattern "Contrarian/Choquant" (le plus viral)
- "Ton médecin te ment sur..."
- "Arrêtez de manger [aliment healthy]"
- "Ce que personne te dit sur..."
- "J'ai perdu X kilos en faisant l'INVERSE"

### Pattern "Storytelling Personnel"
- "J'ai googlé mes symptômes..."
- "Mon médecin m'a dit de..."
- "Ma femme/mon pote de X ans..."

### Pattern "Liste/Chiffres"
- "3 aliments qui font grossir"
- "1 Français sur 3 est..."
- "Le truc à X€ qui change tout"

### Pattern "Comment Bait"
- "Commente ton âge"
- "Commente [MOT-CLÉ] pour recevoir..."
- "Sauvegarde si tu veux pas oublier"
- "Qui est concerné ? 👇"

## ⏱️ DURÉE OPTIMALE
- **15-30 secondes** = sweet spot pour les nouveaux comptes
- Le ratio watch time / durée totale est le signal #1 de l'algo
- Court = plus facile d'avoir 100% de watch time = push algorithmique

## 🎵 MUSIQUE
- Utiliser les sons TENDANCE (pas de BGM random)
- Volume bas sous la voix
- Vérifier le Creative Center TikTok pour les sons qui montent

## 📱 POSTING STRATEGY
- 3 vidéos/jour les 2 premières semaines (nourrir l'algo)
- Poster à 7h, 12h et 19h (pics d'audience FR)
- Répondre à CHAQUE commentaire dans les 30 premières minutes
- Engagement dans les commentaires = signal boost massif
